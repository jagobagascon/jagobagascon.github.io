A bit about myself

Even though I spent most of my career working with web-related technologies I am a very versatile developer.

I can build a highly performant interactive map using JavaScript and an HTML canvas. I can work on the project's CI/CD. And I can also write a custom communication protocol for IoT devices in C.

I can spend several days analysing logs in Wireshark looking for a bug. Or even write a Wireshark dissector plugin to pretty-print a custom wire protocol to make my (and other's) life a little bit better.

I have good problem solving skills. It doesn't matter if it's in the frontend, the backend or DevOps. Wherever there's a challenge I want to be part of the solution.
